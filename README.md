# SWAN Kuleleri - Kart Oyunu

Dijital Oyun Tasarımı öğrencileri tarafından geliştirilmiş kart oyunu uygulaması.

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Sunucuyu başlatın:
```bash
npm start
```

3. Tarayıcıda açın:
```
http://localhost:8888
```

## Özellikler

- 60 kart (1-60 arası)
- 3 seviye: Basit, Orta, Zor
- Kart numarası ile arama
- Otomatik cevap gösterimi
- Modern ve responsive tasarım

## Geliştiriciler

- Firdevs Güngör
- Baran Emir Belhaj
- Mertcan Öncül
